Models manager

"[Models] -> [Manager] menu opens dialog window with created noise calculation models.

Use combo box located at the top of the window to filter types of listed noise source models.

Press [EDIT] button to edit existing noise source model. Noise source model dialog box will appear and you will be able to change model parameters.

Press [DELETE] button to delete existing noise source model. This operation will not be possible if there are any presentations based on selected model.

Press [CLOSE] button to close noise source models manager.
