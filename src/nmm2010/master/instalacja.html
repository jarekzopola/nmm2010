Instalacja

Program udostępniany jest w technologii Java Web Start w związku z czym typowa instalacja nie jest wymagana. Wystarczy wejść na stronę internetową programu i kliknąć przycisk uruchomienia programu. Komputer zadba o pobranie programu, sprawdzenie jego aktualnej wersji i ewentualna aktualizację.

Konfiguracja serwera wydruku

Wybrane prezentacjw wyników badań przygotowywane są na formatach papieru które nie są "fabrycznie" ustawione w serwerze wydruku systemów operacyjnych. Jeżli uzytkownik chce drukować np. prezentację pomiaru całodobowego na arkuszu papieru o wymiarach 914/297mm (wydruk na ploter) konieczne jest dodanie do serwera wydruku ofpowiedniego formularza.

Dodanie formularza w systemie Windows XP

  1. Wybierz START->Drukarki i Faksy
  2. Z menu okna "Drukarki i Faksy" wybierz Plik->Właściwosci serwera
  3. Przejdź do zakładki "Formularze"
  4. Tutaj zdefiniuj nowy formularz (format arkusza papieru) podając następujące parametry:
    • Zaznacz opcję "Nowy formularz"
    • Zaznacz opcję "Jednostki metryczne"
    • Rozmiary arkusza: Wysokość: 91,4cm, Szerokość: 29.7cm
    • Rozmiary marginesów: Lewy: 0,0cm Prawy: 0,0cm Góra: 0,0cm Dół: 0,0cm
